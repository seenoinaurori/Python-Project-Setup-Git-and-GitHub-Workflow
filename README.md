# Python Lab

## Project Overview

Python Lab is a beginner-friendly Python project that demonstrates project setup, Git version control, GitHub workflow, and modular programming. The project uses multiple Python files, a structured directory layout, and Git to track changes throughout development.

## Project Structure

```
python_lab/
│
├── src/
│   ├── main.py
│   ├── utils.py
│   └── config.py
│
├── tests/
│
├── docs/
│   └── README.md
│
└── .gitignore
```

## Features

- Modular Python programming
- Utility functions
  - Square of a number
  - Even/Odd checker
  - Celsius to Fahrenheit converter
  - Personalized greeting
- User input and output
- Git version control
- GitHub repository management
- Feature branch workflow
- Pull request workflow

## Requirements

- Python 3.x
- Git
- GitHub account

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/python-lab.git
```

Navigate to the project:

```bash
cd python_lab/src
```

## Running the Program

Run the application using:

```bash
python main.py
```

or

```bash
python3 main.py
```

## Git Workflow Used

1. Initialize Git repository
2. Create `.gitignore`
3. Make the initial commit
4. Create a feature branch
5. Add new functionality
6. Commit changes
7. Push to GitHub
8. Open a Pull Request
